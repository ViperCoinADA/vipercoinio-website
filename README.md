# ViperCoin.io — Official Website

The official website for **ViperCoin** — a Cardano-based memecoin community.

Built as a single-file static site (pure HTML + CSS + JS). No build step. Open `Website v2.html` in any browser.

---

## Pages

| Page | Description |
|------|-------------|
| Home | Hero, stats, slideshow |
| Lore & Tokenomics | Project backstory, token details |
| Products & Services | Ecosystem products |
| NFTs | NFT collection showcase |
| Memes | 562-image gallery with download |
| About Us | Team cards |
| Contact | Social links |
